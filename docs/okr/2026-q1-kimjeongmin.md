# 김정민 2026 1분기 개인 OKR

> YouTrack KB: REF-A-2566 | 상위: REF-A-2564 (팀원)

## Objective

만권당 서비스 Linux/Kotlin 기반 현대화를 기술적으로 리드하고, 만권당, 투비 서비스를 안정적으로 운영하며 팀 내 개발 문화와 정책을 수립하여 지속 가능한 개발 체계를 구축한다.

## Key Results

### KR1. 만권당 서비스 마이그레이션

- 만권당 전체 서비스 Windows (.NET) → Linux (Spring Boot Kotlin) 현대화 전환 및 재구축 100% 완료
- 기존 Ebook CMS 시스템에서 제공하던 만권당 CMS, 만권당 ADMIN 서비스 분리 완료
- 배포 파이프라인 구축으로 E2E 배포 시간 5분 이내 달성

### KR2. 레거시 데이터베이스 관리 체계 및 AI 기반 코드 품질 확립

- 만권당 (AS-IS), 투비 핵심 SP 100% Git 등록 관리
- SP 변경 프로세스 정책 문서화 완료
- PR 기반 변경 프로세스 준수율 90% 이상 달성
- Claude 기반 코드 리뷰 자동화 도구 구축 및 전체 PR 적용률 100% 달성
- 신규 Kotlin 코드 테스트 커버리지 80% 이상 유지 정책 수립 및 적용
- SonarQube 품질 게이트 통과 기준 정의, Critical/High 0건 유지 프로세스 정착

### KR3. 보안·거버넌스 체계 구현

- 제휴처 단위 DEK 암호화 아키텍처 구현 완료 및 신규 제휴 100% 적용 (누락 0건)
- AWS KMS CMK 기반 암호화 키 관리 체계 구축 완료 및 운영 안정성 100% 유지
- DPA 기술 요건 체크리스트 문서화 100% 완료 및 신규 제휴 계약 사전 검토율 100% 달성
- 보안 정책 미준수로 인한 재작업 발생 0건 유지
- 보안 취약점 자동 검출 프로세스 구축 및 Critical/High 이슈 0건 유지

### KR4. 팀 개발 문화 및 표준 수립

- Kotlin/Spring Boot 코딩 컨벤션 및 프로젝트 구조 표준 문서화
- 핵심 기능(도메인별) 문서화율 80% 달성을 위한 템플릿 및 작성 가이드 배포
- 코드 리뷰 가이드라인 및 PR 머지 정책 문서화
- 온보딩 문서 및 개발 환경 셋업 가이드 완성
