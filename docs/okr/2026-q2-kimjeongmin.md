# 김정민 2026 2분기 개인 OKR (초안)

> 상위: REF-A-3122 (2분기 팀 OKR) | REF-A-2175 (연간 팀 OKR)

## Objective

1분기에 구축한 만권당 현대화 기반을 완결하고, AI 기반 바이브 코딩으로 팀 생산성 도구를 실전 배포하며, B2B 커머스의 데이터 아키텍처를 확정하여 2분기 팀 OKR 달성을 기술적으로 리드한다.

## Key Results

### KR1. 만권당 리눅스 환경 안착 및 인프라 비용 절감 (4월)

> 팀 KR2 연계 — (이월) AWS 인프라 비용 50% 절감 목표 달성

- 1분기 PoC 대상 서비스 100% 리눅스 전환 완료 (잔여 0건)
- 전환 전후 AWS 월 인프라 비용 비교: **50% 이상 절감** 달성 (비용 리포트 첨부)
- 전환 후 4주간 서비스 가용률 **99.5% 이상** 유지, 치명적 장애 **0건**
- 배포 E2E 시간 **5분 이내** 달성 확인

### KR2. AI 바이브 코딩 기반 신규 웹 서비스 개발 및 실전 배포 (5~6월)

> 팀 KR3 연계 — 바이브 코딩으로 업무 효율화 웹 서비스 최소 1건 실사용 환경 배포/운영

- 대상 서비스 선정 및 요구사항 정의 완료: 사용자 스토리 **10개 이상** 도출 (5월 2주차)
- 바이브 코딩 프로세스 적용, 기획~개발~배포 리드타임 **2주 이내** 달성
- 실사용 환경 배포 **1건 이상** 완료 (6월 3주차)
- 배포 후 **2주간** 운영 안정화: 장애 **0건**, 실사용자 **5명 이상** 확보
- 바이브 코딩 프로세스 회고 문서 **1건** 작성 (재현 가능한 프로세스 가이드)

### KR3. B2B 커머스 데이터 스키마 및 권한 격리 아키텍처 확정 (4~5월)

> 팀 KR6 연계 — B2B 전용 데이터 스키마 및 권한 격리 아키텍처 확정

- 멀티 테넌시 데이터 스키마 설계 문서 **1건** 완료 (4월, 테이블 **20개 이상** 정의)
- 권한 격리 아키텍처 설계 문서 **1건** 완료 및 팀 리뷰 통과 (5월 2주차)
- 바자르(상품/주문)·뉴빌링(결제) 연동 API 인터페이스 명세 **100%** 확정 (5월 4주차)
- 외주 개발자 투입용 5W1H 기반 BE 티켓 **15개 이상** 사전 발행
- JEH FE 구조설계와 동기화 미팅 **격주 1회** 이상 수행

### KR4. DPA 템플릿 완성 및 보안 검토 완료 (6월)

> 팀 KR5 연계 — 나루 서비스 활용을 위한 DPA 템플릿 작성 및 정보보안팀 검토 완료

- DPA 기술 요건 체크리스트 항목 **100%** 최종 업데이트
- DPA 템플릿 작성 완료 및 정보보안팀 검토 **승인 1건** 획득 (6월 3주차)
- 신규 제휴 계약 시 DPA 적용 프로세스 문서 **1건** 완성
