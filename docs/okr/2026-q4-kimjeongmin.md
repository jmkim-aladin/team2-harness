# 김정민 2026 4분기 개인 OKR (초안)

> 상위: REF-A-2175 (연간 팀 OKR)

## Objective

B2B 스토어프론트를 런칭 대기 상태로 완성하고, 만권당 문서화·테스트·인프라 비용 절감 등 연간 목표를 마무리하여 초생산성 기술 조직의 기반을 완결한다.

## Key Results

### KR1. B2B 스토어프론트 연동 테스트 완료 및 런칭 대기 (10~11월)

> 연간 KR6 연계 — B2B 필수 기능 개발 완료율 100%, 데이터 정합성 테스트 통과율 100%

- 바자르(상품/주문)·뉴빌링(결제) API 연동 모듈 개발 **100% 완료**
- 통합 테스트 수행 및 데이터 정합성 테스트 **통과율 100%**
- 표준 입점 가이드(API 명세·연동 규격) 문서 **1건** 완성
- 런칭 대기 상태 확인: 신규 파트너 입점 시 즉시 적용 가능 검증 완료

### KR2. 바이브 코딩 1건 개발·배포 (10~11월)

> 연간 KR3 연계 — 바이브 코딩 신규 웹 서비스 (누적 4건)

- 대상 서비스 선정 및 요구사항 정의 완료 (10월 2주차)
- 기획~개발~테스트~배포 리드타임 **4주 이내** 달성
- 실사용 환경 배포 **1건** 완료
- 배포 후 **2주간** 운영 안정화: 장애 **0건**

### KR3. 만권당 핵심 기능 문서화율 80% 달성 (10~11월)

> 연간 KR3 연계 — 주요 서비스별 핵심 기능 문서화율 80% 이상

- 만권당 도메인별 핵심 기능 목록 정의 및 문서화 대상 확정
- 문서화율 **80% 이상** 달성 (기존 작성분 포함)
- 작성 문서 팀 리뷰 **1회 이상** 수행

### KR4. 만권당 MSA 테스트 커버리지 80% 확보 (10~12월)

> 연간 KR3 연계 — 바자르, 만권당 MSA 테스트 코드 커버리지 80% 이상

- 만권당 Kotlin 서비스 대상 테스트 커버리지 현황 측정 및 갭 분석
- 핵심 모듈 우선 테스트 작성, 커버리지 **80% 이상** 달성
- CI 파이프라인에 커버리지 게이트 **적용 완료**

### KR5. AWS 인프라 비용 50% 절감 실적 검증 (12월)

> 연간 KR2 연계 — 월 AWS 인프라 비용 50% 절감

- Q2 리눅스 전환 이후 월별 비용 추이 데이터 수집
- 전환 전 대비 **50% 이상 절감** 달성 여부 검증 보고서 **1건** 작성
