## 자기 회고

### 잘한 업무

- **만권당 정량제 핵심 기능 개발 완료**: B2C/B2B 구분 이용권 판매, 열람 제한, 카테고리 제한, 웰컴쿠폰 등 정량제 서비스에 필요한 기능을 1분기 내에 모두 개발 완료
- **ADMIN MSA 기반 구축 및 문서화**: 만권당 ADMIN을 멀티모듈 구조로 구축하고 개발 환경에 배포. 프로젝트 구조·API 개발·배포 가이드 3종을 작성하여, 기능 구현을 바로 진행할 수 있도록 기반 마련
- **SP Git 버전 관리 체계 구축**: DBSourceTemplate을 개발하여 만권당·투비 P0 SP를 100% Git에 등록하고 운영체제 관계없이 관리, 지속 가능한 체계 구축
- **AASM 서비스 신규 개발**: 서비스팀과 디자인, 개발팀에 모두 필요한 AASM 서비스를 신규 구상하여 AI를 활용하여 빠른 시간 내 설계 및 개발 진행
- **투비 AWS 이관 완료**: 투비 서비스 AWS 클라우드 이관 마무리 기능 점검·모니터링, UCL Editor S3 이관, 검색엔진 변경 등을 수행하여 성공적으로 이관하고 서비스 품질과 사용자 경험 향상을 위해 다양한 개선 진행

### 아쉬운 업무

- **만권당 현대화 개발 지연**: 정량제 개발과 ADMIN 배포, 투비 AWS 이관 등에 리소스가 집중되면서 만권당 현대화 정산 설계(1/4종), ADMIN 핵심기능(0/3건) 진행이 늦어짐

### 기대되는 업무

- **만권당 .NET → .NET Core 전환**: 1분기 PoC에서 전환 가능성을 확인했고, 4월에 실제 전환을 진행. Windows 의존성을 제거하면 Linux 기반 배포 파이프라인과 연결되어 배포 속도와 인프라 비용 모두 개선될 것으로 기대
- **AI 바이브 코딩 실전 적용**: 1분기 AASM 개발 경험을 바탕으로 새로운 시도를 통해 비용절감과 팀 생산성에 기여할 것으로 기대
- **B2B 쇼핑몰 신규 구축**: 다양한 알라딘 서비스의 신규 MSA기능들 (인증, 결제 등) 및 바자르를 통한 주문 연동과 AI를 활용한 설계·개발 및 분업을 시도해볼 수 있어 기대됨
