# OKR 자기평가 작성 가이드

> YouTrack KB: DEV2-A-977 | 상위: DEV2-A-578 (OKR)
> 양식 및 샘플: https://nas.aladin.co.kr/drive/d/f/17fTi9inWgJaFHQoO7YlGS2vN7QjvpX6

## 제출 시점

- 분기 시작 시 전 분기 성과 제출 (예: 2분기 초 → 1분기 평가 제출)

## 시작 전

양식의 아래 항목은 **분기 초 OKR 문서에서 그대로 옮겨 적습니다.**

- Objective
- 핵심 결과 (KR)
- 측정 지표
- 목표치 (측정 지표 내 정량적 숫자)

내용을 바꾸거나 다듬지 말고, 처음 작성한 그대로 옮겨주세요.
평가는 당시 세운 목표 기준으로 이루어집니다.

## 입력 항목

### 1. KR 가중치 제안

분기 초 OKR 문서를 보면서, 이 KR들 중 어디에 더 무게를 뒀는지 돌아보세요.

- 전체 KR 합계가 **60점**이 되도록 배분
- 중요하게 생각했던 KR에 더 높은 점수
- 정수로 입력 (예: 20, 15, 10, 10, 5)

> 팀장이 검토 후 조정할 수 있습니다. 최종 계산은 팀장 조정값 기준입니다.

### 2. 지표 가중치 제안

각 KR 안에서 측정지표들의 중요도를 비율로 표현합니다.

- KR별로 측정지표 합계가 **100%**가 되도록 입력
- 핵심으로 생각했던 지표에 더 높은 비중
- 정수로 입력 (예: 70, 30)

> 팀장이 조정할 수 있습니다. 본인 판단을 그대로 적어주세요.

### 3. 달성치

분기 초에 세운 목표치 옆에, 실제로 달성한 수치를 입력합니다.

- 목표치와 **동일한 단위**로 입력
- 달성률과 득점은 자동 계산됩니다

### 4. 성과 상세 근거

달성치를 뒷받침하는 근거를 적습니다.

- 티켓, 커밋, 문서 중에서 지표와 직접 연결되는 것을 선택
- 티켓이 없으면 배포 이력, 문서 링크, 스프린트 기록 등
- 한 줄에 하나씩

> 근거 없는 달성치는 인정되지 않을 수 있습니다.

### 5. 자기 회고

| 항목 | 작성 방향 |
|------|-----------|
| 잘한 업무 | 잘 됐다고 생각하는 것과 그 이유 |
| 아쉬운 업무 | 기대에 못 미쳤던 부분과 원인 |
| 기대되는 업무 | 다음 분기에 이어가거나 새로 도전하고 싶은 것 |

## 자동 계산 항목 (입력 불필요)

| 항목 | 계산 방식 |
|------|-----------|
| 달성률 | 달성치 ÷ 목표치 × 100 |
| 구간 배율 | 달성률 구간에 따라 자동 적용 |
| 지표 득점 | KR 가중치 × 팀장 조정 가중치% × 구간 배율 |
| KR 득점 | 해당 KR의 지표 득점 합계 |
| OKR 총점 | 전체 KR 득점 합계 (만점 60점) |

## 주의사항

1. 노란색 셀만 입력합니다.
2. KR과 측정지표는 분기 초 문서 내용 그대로 옮깁니다. 수정하지 마세요.
3. KR 가중치 합계가 60점인지 확인하세요.
4. 지표 가중치는 KR별 합계 100%가 되도록 입력하세요.
5. 달성치는 목표치와 동일한 단위로 입력하세요.
6. 성과 근거는 티켓 번호나 링크로 구체적으로 적어주세요.

## 팀장 입력 영역 (팀원 작성 불필요)

제출 후 팀장이 입력합니다.

- KR / 지표 가중치 조정
- OKR 최종 점수
- 핵심역량 점수 (40점 만점)
- 가산 / 감점
- 팀장 피드백
