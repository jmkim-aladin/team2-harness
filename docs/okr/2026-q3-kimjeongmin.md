# 김정민 2026 3분기 개인 OKR (초안)

> 상위: REF-A-3122 (2분기 팀 OKR) | REF-A-2175 (연간 팀 OKR)

## Objective

B2B 스토어프론트 핵심 기능 개발을 본격 착수하고, 만권당 CMS/ADMIN 운영 배포와 보안·품질 체계를 정착시켜 연간 목표 달성의 실행 전환점을 만든다.

## Key Results

### KR1. B2B 스토어프론트 핵심 기능 개발 (7~9월)

> 연간 KR6 연계 — B2B 필수 기능(멀티 테넌시 전시·주문) 엔진 개발

- 2분기 확정된 Aggregate 기반 도메인 모델 구현 완료
- 전시·주문 엔진 핵심 API **개발 완료율 70% 이상** 달성
- 바자르(상품/주문) 연동 모듈 개발 착수 및 **핵심 API 3개 이상** 연동 완료
- 개발 모듈 유닛 테스트 커버리지 **80% 이상** 유지

### KR2. 바이브 코딩 1건 개발·배포 (7~8월)

> 연간 KR3 연계 — 바이브 코딩 신규 웹 서비스 (누적 3건, Q1 1건 + Q2 1건 + Q3 1건)

- 대상 서비스 선정 및 요구사항 정의 완료 (7월 2주차)
- 기획~개발~테스트~배포 리드타임 **4주 이내** 달성
- 실사용 환경 배포 **1건** 완료
- 배포 후 **2주간** 운영 안정화: 장애 **0건**

### KR3. 만권당 CMS/ADMIN 추가 개발 및 운영 배포 완료 (7~8월)

> 연간 KR5 연계 — 만권당 CMS/ADMIN 서버 및 시스템 분리 완료

- 운영 배포 전 추가 개발 항목 식별 및 **100% 완료**
- 운영 환경 배포 완료 (8월 내)
- 배포 후 **2주간** 가용률 **99.5% 이상**, 치명적 장애 **0건**

### KR4. DPA 정보보안팀 승인 획득 (7월)

> 연간 KR5 연계 — 나루 서비스 DPA 이행률 100%

- 2분기 검토 요청 건 정보보안팀 승인 **1건** 획득
- 신규 제휴 계약 시 DPA 적용 프로세스 **1회 이상** 실적용

### KR5. SonarQube 전체 서비스 적용 및 월 검사 체계 구축 (8~9월)

> 연간 KR5 연계 — SonarQube 취약점 검사 적용, Critical/High 0건 유지

- max, tobe, naru, aasm **4개 서비스** SonarQube 파이프라인 연동 완료
- 월 1회 검사 보고서 작성 체계 수립 및 **1회 이상** 실행
- Critical/High 이슈 **0건** 유지
