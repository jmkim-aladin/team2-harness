# 개발2팀 2026 연간 OKR

> YouTrack KB: REF-A-2175 | 작성자: 김규태

## Objective

바자르 고도화와 만권당 마이그레이션을 완수하여 영업이익 증대에 기여하며, AI 기반 자동화와 보안·거버넌스 강화를 통해 초생산성 기술 조직으로 도약한다.

## Key Results

### KR 1. 바자르 매출 확대

- 6개의 오픈마켓과 연동 완료
- 오픈마켓과 연관된 주문 / CS / 교반품 처리를 바자르에서 가능하도록 일원화하여, 네이버스마트스토어 주문 기준 현행 반품처리 CS 평균 소요시간 30분 → 5분 이내 단축(약 83% 감소)

*KR 기여자:*
- 안혜련(KR1, 2, 3)
- 이현민(KR4)
- 조윤주(KR1)

### KR 2. 만권당 인프라 비용 최적화

- 운영체제를 윈도우즈에서 리눅스로 마이그레이션 완수하여, 월 AWS 인프라 비용 50% 절감
- 배포 엔드투엔드 시간: 평균 40분 → 5분 이내 단축(약 87.5% 감소)

*KR 기여자:*
- 김정민(KR1)
- 조은흠(KR1)

### KR 3. AI 기반 생산성 강화

- 바자르, 만권당 MSA의 테스트 코드 커버리지 80% 이상 유지
- 주요 서비스별 핵심 기능 문서화율 80% 이상 달성
- AI 에이전트 기반 '바이브 코딩'을 활용한 신규 웹 서비스(업무 효율화 및 서비스 지원 등) 3건 초고속 개발 및 실사용 환경 배포/운영 **(2분기 추가)**

*KR 기여자:*
- 김정민(KR2, KR4)
- 이현민(KR1, KR2, KR3)
- 조윤주(KR2, KR3)
- 이유림(O1, O3)
- 조은흠(KR3)

### KR 4. 고객 경험 및 디자인 일관성 강화

- 만권당 신규 화면 개발 시, 디자인 시스템 컴포넌트 기반 시안 적용율 95% 이상 달성
- 신규 기능 화면 배포 시, 디자인 시안과 구현 화면간 일치율 100% 유지
- SDUI 기반 이벤트 관리 시스템을 만권당과 투비 서비스에 적용 **(2분기 추가)**

*KR 기여자:*
- 김정실(O1, O2, O4)
- 이유림(O2)
- 조은흠(KR2)

### KR 5. 보안·거버넌스 체계 강화

- 신규 제휴 계약 시 DPA 명시화 이행률 100%
- 신규 제휴 데이터는 제휴처 단위 DEK 암호화 + AWS KMS 기반 CMK 관리로 100% 암호화·분리 보관 달성
- SonarQube 취약점 검사 적용 → Critical/High 이슈 0건 유지
- 만권당 CMS / ADMIN 서버 및 시스템 분리 완료

*KR 기여자:*
- 김정민(KR2, KR3)
- 안혜련(KR4)

### KR 6. B2B 전용 커머스(스토어프론트) 개발 및 연동 완료 (런칭 대기) (2분기 추가)

- B2B 필수 기능(멀티 테넌시 대응 전시, 주문, 정산) 엔진 개발 완료율 100%
- 바자르(상품/주문) 및 뉴빌링(결제) API 연동 모듈 개발 및 데이터 정합성 테스트 통과율 100%
- 신규 B2B 파트너 입점 시 즉시 적용 가능한 '표준 입점 가이드(API 명세 및 연동 규격)' 완성
